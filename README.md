# Oleksandra - Personal Portfolio

A one-page portfolio website built with SvelteKit 5, optimized for SEO and mobile responsiveness.

## Features

- ✅ SvelteKit 5 with SSR (Server-Side Rendering) for SEO
- ✅ Tailwind CSS for styling
- ✅ Fully responsive design
- ✅ Optimized for Google crawlers
- ✅ Ready for Netlify deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for Netlify deployment. Simply:

1. Push to GitHub
2. Connect your repository to Netlify
3. Netlify will automatically detect the SvelteKit project and deploy

## Customization

- Update profile image: Replace `/static/profile.jpg`
- Update contact links: Edit the Telegram and LinkedIn URLs in `src/routes/+page.svelte`
- Customize colors: Modify `tailwind.config.js`
